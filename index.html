Our projects

Quick Starter

3 different distrobutions,
Quickstarter is focused on making your life easier by starting an application with just one command

BLUETOOLS

BLUETOOLS is a Command-Line Interface, which allows you to do various actions, that may make your life easier.

SISHELL

SISSHELL is inspired by
The Windows CMD Shell and Bash, it provides more color, shorter syntaxes and a login system, which can be toggled.

PackIO

A place for downloading and uploading Minecraft resourcepacks, comes with an Ingame Modification, which allows you to easily install resourcepacks, it also offers a Rest API for developers to download resourcepacks.

Winfo

Python Library to get information about Windows Systems (10/11). It provides CPU, GPU, Memory, Disk, Ethernet, Motherboard and Software information.
Simple installation with pip:
"pip install Winfo"
